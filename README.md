## 本地开发

### 环境准备

- 安装 [Node.js](https://nodejs.org/en)
- 安装 [pnpm](https://pnpm.io/installation)

### 操作步骤

- 安装依赖

```sh
pnpm install
```

- 启动 Dev Server

```sh
pnpm run dev
```

- 在浏览器访问 http://localhost:3200

## 访问地址说明

项目配置了两个访问地址，指向同一个页面但具有不同的权限：

- **大众版（无编辑权限）**: http://localhost:3200
  - 面向普通用户，无法编辑组件元素
  
- **内部版（有编辑权限）**: http://localhost:3200/admin
  - 面向内部人员，可以点击组件元素打开编辑窗口进行修改

## 快捷启动

在项目根目录下，双击 `启动项目.bat` 文件即可快速启动项目（需要已安装 Node.js 和 pnpm）。

该脚本会自动：
- 检查 Node.js 和 pnpm 是否已安装
- 如果未安装 pnpm，会自动尝试安装
- 如果是首次运行，会自动安装依赖
- 启动开发服务器

## 编辑功能说明

在内部版（/admin）中，您可以：
- 点击页面上的组件元素（如 Logo、轮播图、新闻等）打开编辑窗口
- 修改组件的内容、图片、链接等信息
- 保存修改后的内容会存储在浏览器的本地存储中
